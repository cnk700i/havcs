## 调试Tips
1. 根据[教程][3]调整插件的调试级别查看详细的运行日志
2. 配置好插件，启动HA，观看是否有mqtt连接成功信息（app技能接入方式）
3. 授权过程，观看是否有相关的处理日志
4. 说音响指令后，观看是否有相关的处理日志
5. web页面->开发者工具->服务->havcs.reload，观看是否生成设备信息
> WARN: 如需要帮助，请提供以上步骤相对应的日志信息方便定位原因。
